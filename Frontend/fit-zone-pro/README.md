# 🏋️ FitZone Pro — E-Commerce Website

FitZone Pro is a premium, high-performance, single-page e-commerce website built for a leading fitness equipment brand in Pakistan. It features a modern dark-themed UI with high-impact typography, glassmorphism effects, and a fully functional shopping cart system.

## ✨ Key Features

- **📱 Fully Responsive**: Optimized for all devices using a 3/2/1 grid layout.
- **🛒 Dynamic Cart System**: Add to cart, adjust quantities, and clear cart (persisted via `localStorage`).
- **💬 WhatsApp Integration**: Real-time checkout where users can send their order details directly to WhatsApp.
- **🔍 Product Filtering**: Filter equipment by categories: Strength, Cardio, and Yoga.
- **⚡ Quick View**: Instantly view product details and specifications in a sleek glassmorphic modal.
- **🌓 Dark Mode**: Built-in toggle to switch between high-contrast Dark and Light themes.
- **📊 Fitness Tools**:
    - **BMI Calculator**: Interactive tool for users to check their health stats.
    - **Workout Generator**: Personalized workout plans based on goal and experience level.
- **🔥 Conversion Boosters**:
    - **Countdown Timer**: 3-day flash sale timer.
    - **Toast Notifications**: Interactive feedback for user actions.
    - **Animated Stats**: Real-time counters for customers and products sold.
    - **Trust Marquee**: Infinite scrolling bar showcasing brand reliability.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **TailwindCSS**: For modern, utility-first styling.
- **Vanilla JavaScript**: 100% framework-free logic.
- **Font Awesome 6**: Premium iconography.
- **Google Fonts**:
    - *Bebas Neue*: For aggressive, high-impact headings.
    - *Inter*: For clean, professional body text.

## 🛠️ Setup Instructions

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/fit-zone-pro.git
   cd fit-zone-pro
   ```

2. **Add Assets**:
   Create an `assets/` folder and add the required images (logo, hero, products, and gallery).

3. **Run Locally**:
   Simply open `index.html` in any web browser. No local server or build process is required!
